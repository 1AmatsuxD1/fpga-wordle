# Archive

ไฟล์ที่ไม่ได้ใช้งานในโปรเจ็กต์หลักอีกต่อไป

## Files

### serial_transmitter_simple.vhd
- เวอร์ชัน debug ของ serial_transmitter
- ส่ง clock ออกตลอดเวลา (free-running) เพื่อทดสอบ hardware
- ไม่ได้ใช้ในโปรเจ็กต์สุดท้าย (ใช้เวอร์ชันปกติแทน)

### serial_communication_tb.vhd
- Testbench สำหรับทดสอบ serial communication
- ไม่ได้ใช้ใน hardware จริง

### STATUS_CHECK_2025-11-03.md
- เอกสารเช็คสถานะโปรเจ็กต์เมื่อวันที่ 3 พฤศจิกายน 2025
- ข้อมูลล้าสมัย

### Debug Documentation Files
- `DEBUG_CHECKLIST.md` - รายการเช็คปัญหา
- `DEBUG_LED_GUIDE.md` - คู่มือ LED debug สำหรับ FPGA #2
- `DEBUG_LED_GUIDE_FPGA1.md` - คู่มือ LED debug สำหรับ FPGA #1
- `DEBUG_MAGENTA_ISSUE.md` - คู่มือแก้ไขปัญหาหน้าจอสีม่วง
- เอกสาร debug ชั่วคราวที่สร้างระหว่างการแก้ไขปัญหา
